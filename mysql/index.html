    MySQL 数据库笔记¶

    全面的 MySQL 学习笔记，涵盖从基础到进阶的核心知识点。

    目录¶

    章节 标题 内容概述
    01 基础与 DDL 数据库概念、安装配置、数据库/表操作
    02 DML 与查询 增删改查、SELECT、JOIN、子查询、聚合
    03 索引与优化 索引类型、原理、Explain、SQL 优化
    04 事务与锁 事务 ACID、隔离级别、各种锁机制
    05 高级特性 存储引擎、视图、存储过程、触发器、函数
    06 运维管理 用户权限、备份恢复、日志、主从复制

    MySQL 版本¶

    这些笔记基于 MySQL 8.0 编写，大部分内容适用于 MySQL 5.7+。

    学习路线¶

    基础概念 → DDL/DML → 查询进阶 → 索引优化 → 事务锁 → 高级特性 → 运维管理

    建议按顺序阅读，每个章节都可以独立参考。
