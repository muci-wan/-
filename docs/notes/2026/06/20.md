## 学习笔记

### MkDocs + GitHub Pages

MkDocs 是一个快速的静态站点生成器，专为项目文档设计。配合 Material for MkDocs 主题，可以搭建出带有搜索、深色模式的美观文档站点。

**搭建步骤：**

1. 创建 `mkdocs.yml` 配置文件
2. 在 `docs/` 目录放置 Markdown 文件
3. 用 GitHub Actions 自动构建部署
4. GitHub Pages 提供免费托管
