## MySQL 版本

这些笔记基于 **MySQL 8.0** 编写，大部分内容适用于 **MySQL 5.7+**。

```
基础概念 → DDL/DML → 查询进阶 → 索引优化 → 事务锁 → 高级特性 → 运维管理
```

建议按顺序阅读，每个章节都可以独立参考。
